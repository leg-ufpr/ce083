Recursos computacionais

Os recursos computacionais físicos nos laboratórios de aula são mantidos pelo C3SL (Centro de Computação Científica e Software Livre). Para acesso ao sistema e dicas computacionais leia: Uso do sistema computacional do C3SL.

Linux

Os recursos computacionais mantidos pelo C3SL são todos baseados em Linux. Alguns materiais que podem ser consultados são Introdução ao uso do Linux e Introdução ao shell Linux.

R

O R é um programa para análise estatística de livre distribuição e de código aberto. Isso significa que você pode baixar, instalar, repassar para seus amigos e até mesmo alterar o código fonte de acordo com suas necessidades. Ele pode ser instalado em diversas arquiteturas e sistemas operacionais diferentes, incluindo Windows, Linux, Macintosh e outros.

Para instalar o R, veja este guia de instalação.
